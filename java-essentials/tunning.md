# JMC, JFR
### JFR Java Flight Recorder 飞行记录仪
垃圾回收器的基本假设：
* 飞行记录仪主要用于记录JVM内部的事件
* JFR可以记录到文件，然后由JMC读取，展示
* JMC是基于JMX的控制台，展示JFR记录的数据
